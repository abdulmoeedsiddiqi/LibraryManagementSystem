# 📚 Library Management System (Java MVC)

A structured and scalable **Library Management System** developed in **Java** using the **MVC (Model-View-Controller) architecture**. This project demonstrates clean code practices, modular design, and effective separation of concerns for managing library operations.

## 🚀 Features
- 📖 Add, update, delete, and view books  
- 👤 Manage users/members  
- 🔄 Issue and return books  
- 🔍 Search books by title, author, or ID  
- 📊 Maintain records of transactions  

## 🏗️ Architecture
This project follows the MVC design pattern:
- **Model:** Manages data and core logic (Book, User, Transaction)  
- **View:** Displays information to the user (Console/GUI)  
- **Controller:** Handles user input and connects Model with View  

## 🛠️ Technologies Used
- Java  
- Object-Oriented Programming (OOP)  
- MVC Architecture  

## 📂 Project Structure
Library-Management-System/  
├── model/        # Data classes  
├── view/         # UI layer  
├── controller/   # Application logic  
└── Main.java     # Entry point
